# Changelog

## Latest Updates

### Video & Content Improvements
- ✅ **Larger Video Player**: Increased YouTube iframe size to minimum 500px height for better viewing experience
- ✅ **YouTube URL Conversion**: Automatic conversion of YouTube watch URLs and short URLs to embed format
- ✅ **PDF Same-Tab Viewing**: PDFs now display in same tab for both students and instructors
- ✅ **File Upload Support**: Added file upload functionality for videos and PDFs in course modules

### Progress Tracking Enhancements
- ✅ **Automatic Progress Updates**: Progress automatically updated to 100% when students view modules
- ✅ **Visual Completion Indicators**: 
  - Green "Completed" badges on finished modules
  - Checkmark icons replace module numbers for completed modules
  - Progress bars on dashboard
- ✅ **Real-time Updates**: Progress updates immediately when modules are viewed

### Course Discovery Features
- ✅ **Search Functionality**: Real-time search across course titles, descriptions, and instructors
- ✅ **Instructor Filtering**: Filter courses by instructor with dropdown selector
- ✅ **Pagination**: Browse courses with pagination (9 courses per page)
- ✅ **Smart Messaging**: "No courses found" message when search/filter returns no results

### Question & Answer System
- ✅ **Student Questions**: Students can ask questions on course pages
- ✅ **Instructor Answers**: Instructors can respond to questions
- ✅ **Q&A Display**: All questions and answers shown with timestamps and user names

### Security & UX Improvements
- ✅ **Password Validation**: Strong password requirements (uppercase, lowercase, number, underscore)
- ✅ **Password Visibility Toggle**: Eye icon to show/hide passwords
- ✅ **Real-time Validation Feedback**: Visual indicators for password requirements

### UI/UX Enhancements
- ✅ **Learning Theme Colors**: Updated to pleasant blue and green color scheme
- ✅ **Performance Optimization**: Reduced animations for faster page loads
- ✅ **Modern Design**: Clean, professional interface optimized for learning

## Previous Features
